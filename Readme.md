# Análisis de E-Commerce Brasileño (Olist)

## Objetivo del Proyecto
Analizar el rendimiento de ventas y logística de Olist Store utilizando SQL para limpiar y estructurar datos, y Power BI para visualizar KPIs clave.

## Estructura de Datos
El dataset contiene 100k pedidos realizados entre 2016 y 2018 en múltiples marketplaces de Brasil.

## Herramientas
* SQL (SQLite)
* DBeaver
* VS Code
